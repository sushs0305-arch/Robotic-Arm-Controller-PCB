# Robotic-Arm-Controller-PCB
Custom STM32-based robotic arm controller PCB designed using KiCad with servo motor control, joystick interface, OLED support, Bluetooth connectivity, and USB Type-C power input.

## Overview

This project is a custom-designed STM32-based PCB developed for robotic arm control applications.  
The PCB was designed using KiCad and integrates multiple peripheral interfaces including servo motor outputs, joystick control, OLED display connectivity, Bluetooth communication, and USB Type-C power input.

The design focuses on compact PCB layout, embedded hardware integration, and efficient routing for robotics applications.

---

## Features

- STM32F103CBT6 Microcontroller
- 4 Servo Motor Interfaces
- Joystick Connector
- HC-05 Bluetooth Module Interface
- OLED Display Support
- USB Type-C Power Input
- AMS1117 3.3V Voltage Regulation
- Compact 2-Layer PCB Design

---

## Components Used

| Component | Description |
|---|---|
| STM32F103CBT6 | ARM Cortex-M3 MCU |
| HC-05 | Bluetooth Module |
| OLED Display | Display Interface |
| AMS1117-3.3 | Voltage Regulator |
| Servo Headers | Servo Motor Connections |
| USB Type-C | Power Supply |
